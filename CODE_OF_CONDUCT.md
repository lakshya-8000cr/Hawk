# Code of Conduct

## Our Commitment

We are committed to maintaining a welcoming, respectful, and inclusive community for everyone contributing to Hawk.

Participants are expected to communicate professionally, respect differing viewpoints, and collaborate constructively.

## Expected Behavior

- Be respectful and considerate.
- Provide constructive feedback.
- Focus discussions on technical topics.
- Help create a positive environment.

## Unacceptable Behavior

Examples include:

- Harassment or discrimination.
- Personal attacks.
- Trolling or inflammatory comments.
- Publishing private information.
- Any behavior that creates an unsafe environment.

## Enforcement

Project maintainers are responsible for enforcing this Code of Conduct and may remove comments, issues, or contributions that violate these guidelines.

## Scope

This Code of Conduct applies to all project spaces, including:

- GitHub Issues
- Pull Requests
- Discussions
- Documentation
- Community interactions

Thank you for helping make Hawk a welcoming project for everyone.
