## [v1.0.0] - Initial Release

### Added

- Native `kubectl` plugin implementation.
- Dependency discovery for Deployments, ReplicaSets, Pods, Services, ConfigMaps, Secrets, PersistentVolumeClaims, and Ingresses.
- In-memory dependency graph construction.
- Blast Radius evaluation.
- Read-only Kubernetes API interaction.
- Cross-platform builds.
- Krew installation support.
- Comprehensive project documentation.
